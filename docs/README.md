# Documentation Index

This folder contains practical setup and usage notes for working with the local project and the Deep-Live-Cam workflow.

## Documents

- [GitHub Desktop workflow](GITHUB_DESKTOP.md) — clone, commit, push, and sync using GitHub Desktop.
- [Local setup](LOCAL_SETUP.md) — Python environment, dependencies, and model placement.
- [OBS setup](OBS_SETUP.md) — using OBS Studio as a camera/stream source.
- [Troubleshooting](TROUBLESHOOTING.md) — common runtime, dependency, and camera issues.

## Recommended order

1. Start with [GitHub Desktop workflow](GITHUB_DESKTOP.md) if you want to manage the repository from the desktop app.
2. Follow [Local setup](LOCAL_SETUP.md) to install Python packages and models.
3. Use [OBS setup](OBS_SETUP.md) when you want OBS as your camera or streaming layer.
4. Refer to [Troubleshooting](TROUBLESHOOTING.md) if the app does not start correctly.
