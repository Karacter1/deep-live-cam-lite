# OBS Studio setup

OBS Studio can be used as a camera or streaming layer for your workflow.

## 1. Install OBS Studio

Download and install OBS Studio from the official website.

## 2. Enable the virtual camera

- Open OBS Studio.
- In the Sources panel, add your camera or screen capture source.
- Start the virtual camera from OBS.

## 3. Use OBS with the project

Depending on your workflow, OBS can be used in one of two ways:

1. As a camera source for your live feed.
2. As an intermediate layer before streaming or recording.

## 4. Recommended approach

For the smoothest experience:

- Run the Deep-Live-Cam app on your local desktop.
- Start OBS first.
- Route the camera feed through OBS.
- Use the OBS feed as the input or preview source where needed.

## Notes

OBS is especially useful if you want to manage camera switching, overlays, or scene transitions before the final stream or capture.
