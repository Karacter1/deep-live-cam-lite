# GitHub Desktop workflow

GitHub Desktop can be used to clone, pull, commit, and push this repository on your PC.

## 1. Clone the repository

- Open GitHub Desktop.
- Choose File > Clone repository.
- Enter the repository URL:
  - https://github.com/Karacter1/karadfl.git
- Choose a local folder on your computer and click Clone.

## 2. Open the project folder

- After cloning, open the repository in GitHub Desktop.
- Use the Show in Explorer / Open in Finder button to open the local folder.

## 3. Work with the project

- Make changes in the project folder.
- In GitHub Desktop, review the changed files.
- Enter a commit message such as:
  - Add documentation and setup notes
- Click Commit to main (or create a branch if you want to work safely).

## 4. Push changes

- Click Push origin to upload the commit to GitHub.
- If prompted, sign in to GitHub in the desktop app.

## 5. Pull updates

- Use Pull origin to fetch updates from GitHub before you continue working.

## Notes

GitHub Desktop is a convenient way to manage the repository, but it does not replace the need to install Python dependencies and runtime libraries locally.
