## Recommended workflow

1. Use GitHub Desktop if you want a simple GUI for cloning, committing, and pushing changes.
2. Follow the local setup guide to install Python dependencies and place the required models.
3. Use OBS Studio if you want to route your camera or stream through a desktop workflow.
4. Refer to the troubleshooting guide if the app does not launch correctly.
